== About This Documentation

This documentation describes the architecture of *DailyStandApp*, a **privacy-first** Kotlin Multiplatform application that helps developers prepare for daily standup meetings by analyzing git commits and generating AI-powered summaries.

IMPORTANT: *Privacy is the core principle.* All AI processing runs locally using SKaiNET kllama. Git commits contain sensitive information - no data ever leaves the developer's machine.

The documentation follows the https://arc42.org[arc42] template for software architecture documentation.

== Technology Stack

* *Language*: Kotlin 2.2+ with Multiplatform
* *Runtime*: JDK 21+ (required for Vector API)
* *Build System*: Gradle with Kotlin DSL
* *AI Engine*: SKaiNET kllama
* *Git Integration*: Eclipse JGit
* *Protocol*: Model Context Protocol (MCP) SDK
